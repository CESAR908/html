## ✨ Características

### 🏗️ Estrutura Semântica
- HTML5 com tags semânticas (`<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<footer>`)
- Hierarquia de headings correta
- Uso apropriado de `<figure>` e `<figcaption>`
- Elementos `<blockquote>` para citações

### ♿ Acessibilidade (WCAG 2.1 AA)
- Atributos `alt` descritivos em todas as imagens
- Atributos `aria-label` para navegação
- Contraste de cores adequado
- Suporte a navegação por teclado
- Compatibilidade com leitores de tela
- Links com texto descritivo

### 📱 Design Responsivo
- Grid layout fluido
- Flexbox para navegação
- Media queries para mobile, tablet e desktop
- Imagens responsivas
- Tipografia escalável

### 🎨 Design Moderno
- Paleta de cores profissional
- Layout limpo e intuitivo
- Barra lateral com informações rápidas
- Índice navegável
- Cronologia visual

## 🚀 Quick Start

### 1️⃣ Clonar o Repositório
```bash
git clone https://github.com/seu-usuario/revolucao-francesa.git
cd revolucao-francesa
```

### 2️⃣ Abrir no Navegador
```bash
# Opção 1: Abrir diretamente
open index.html

# Opção 2: Usar servidor local (Python 3)
python -m http.server 8000
# Acesse: http://localhost:8000

# Opção 3: Usar Live Server (VS Code)
# Instale a extensão "Live Server" e clique em "Go Live"
```

### 3️⃣ Editar o Conteúdo
```bash
# Abra em seu editor favorito
code .  # VS Code
vim index.html  # Vim
```
